プライバシーポリシー

アプリ名: Lite Memo（com.appvoyager.litememo）
最終更新日: 2026年6月3日

[開発者名]（以下「当方」といいます）は、Android アプリ「Lite Memo」（以下「本アプリ」といいます）における利用者の情報の取り扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます）を定めます。

1. 収集する情報

1.1 利用者がアプリ内で作成する情報（メモの内容）

利用者が本アプリで作成・編集したメモやその関連データは、利用者の端末内にのみ保存されます。当方はこれらの内容を取得・閲覧・外部サーバーへ送信することはありません。

1.2 自動的に収集される情報（第三者サービス経由）

本アプリは、広告の配信および不具合の解析のため、以下の第三者サービスを利用しています。これらのサービスは、その提供者のプライバシーポリシーに基づき情報を収集する場合があります。

サービス目的収集され得る情報の例
Google AdMob
（広告配信）		 本アプリ内での広告表示 広告識別子（Advertising ID）、デバイス情報、おおよその位置情報（IP アドレスに基づく）、広告の表示・操作に関する情報
Firebase Crashlytics
（不具合解析）		 アプリの安定性向上・不具合の特定と修正 クラッシュ時のログ、端末の状態、OS バージョン、デバイス種別などの診断情報

2. 利用目的

  • 本アプリの提供および機能の維持
  • 広告の表示（本アプリの運営のため）
  • 不具合の検出・解析および品質向上

3. 第三者への提供・情報の共有

本アプリは、上記第三者サービスの提供者に対して、サービス提供に必要な範囲で情報を共有します。各サービス提供者による情報の取り扱いについては、以下をご参照ください。

  • Google のプライバシーポリシー: https://policies.google.com/privacy
  • Google 広告における情報の使用について: https://policies.google.com/technologies/ads
  • Firebase のプライバシーとセキュリティ: https://firebase.google.com/support/privacy

4. 広告について

本アプリは Google AdMob による広告を表示します。広告のパーソナライズや広告識別子の利用は、端末の設定から制御できます。

  • Android の場合: 「設定」→「Google」→「広告」から、広告 ID のリセットまたはパーソナライズ広告のオプトアウトが可能です。

5. データの保管と削除

本アプリで作成したメモは端末内に保存され、本アプリのバックアップ機能は無効化されています。本アプリをアンインストールすると、端末内に保存されたメモは削除されます。第三者サービスが収集した情報の保持期間および削除については、各提供者のポリシーに従います。

6. お子様のプライバシー

本アプリは、13 歳未満（または各国・地域で定められる年齢未満）のお子様を主たる対象としていません。お子様の個人情報を意図的に収集することはありません。

7. 端末の権限について

本アプリは、広告配信および不具合解析に必要な範囲で、利用する第三者サービスが要求する権限（広告 ID へのアクセス、ネットワーク接続など）を使用する場合があります。

8. 本ポリシーの変更

当方は、必要に応じて本ポリシーを変更することがあります。変更後の内容は本ページに掲載した時点から効力を生じるものとします。
